# speedsterAdminWeb

一、项目概述

本系统为一个多端外卖平台的管理后台，服务于平台运营人员，用于统一管理用户、商家、订单、配送、财务及系统配置等核心业务。

二、系统模块总览

- 仪表盘
- 用户管理
- 商家管理
- 商品管理
- 订单管理
- 配送管理
- 财务管理
- 营销中心
- 权限管理
- 系统设置
